## Configuration Guide

To get started, simply edit the **Profile** and **Links** sections in `config.js`. Once updated, your setup will be ready to go!

### Link Properties

Each link comes with the following customizable properties:

- **Title** â€“ The visible text on your link's button.  
- **Description** â€“ A short description displayed beneath the title.  
- **Extend** â€“ If this field is filled, the button transforms into an accordion. When expanded, it reveals this content along with the link button.  
- **URL** â€“ The destination visitors are redirected to when clicking the button.  
- **Color** â€“ The buttonâ€™s color, supporting HEX codes.  
- **Icon** â€“ The icon for your link. Use a Font Awesome class name, which you can find on the [Font Awesome website](https://fontawesome.com).
